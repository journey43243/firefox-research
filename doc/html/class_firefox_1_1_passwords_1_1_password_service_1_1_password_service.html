Подробное описание

Высокоуровневый сервис извлечения паролей Firefox.

Класс выполняет чтение сохранённых учётных данных Firefox из файлов профиля и использует NSSWrapper для их расшифровки.

Поддерживаются следующие форматы хранения:

  • logins.json (актуальный формат);
  • signons.sqlite (устаревший формат).

◆ get_passwords()

List[Dict[str, str]] Firefox.Passwords.PasswordService.PasswordService.get_passwords ( self)

Извлекает и расшифровывает сохранённые пароли Firefox.

Последовательно:

  • инициализирует NSS для профиля;
  • выполняет аутентификацию;
  • считывает сохранённые логины;
  • дешифрует имя пользователя и пароль;
  • завершает работу NSS.
Возвращает
Список словарей со следующими ключами:
  • url — адрес сайта;
  • user — имя пользователя;
  • password — расшифрованный пароль.
