Подробное описание

Обёртка над библиотекой NSS (Network Security Services).

Класс инкапсулирует низкоуровневую работу с библиотекой NSS и предоставляет удобный интерфейс для:

  • загрузки динамической библиотеки NSS;
  • инициализации профиля Firefox;
  • аутентификации пользователя (мастер-пароль);
  • дешифрования данных PKCS#11;
  • корректного освобождения ресурсов NSS.

Используется для расшифровки конфиденциальных данных Firefox, включая сохранённые логины и пароли.

Дешифрует строку Base64, зашифрованную средствами NSS.

Использует функцию PK11SDR_Decrypt для расшифровки данных, закодированных в формате Base64.
