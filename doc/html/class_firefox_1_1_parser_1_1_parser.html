Подробное описание

Управляет процессом извлечения данных Firefox.

Класс Parser координирует работу стратегий извлечения данных браузера Firefox.

В рамках одного кейса:

  • определяет пути профилей Firefox;
  • инициализирует стратегии обработки данных;
  • передаёт им метаданные профиля;
  • запускает выполнение стратегий;
  • обеспечивает сохранение результатов в SQLite.

Используется как точка входа для анализа данных Firefox.

Аргументы
parametersСловарь параметров кейса, содержащий:
  • LOG: интерфейс логирования;
  • CASEFOLDER: путь к каталогу кейса;
  • CASENAME: имя кейса;
  • DBCONNECTION: интерфейс соединения с БД;
  • OUTPUTFILENAME: имя результирующего файла БД;
  • OUTPUTWRITER: интерфейс записи данных;
  • MODULENAME: имя модуля (Firefox).

Методы

◆ Start()

Firefox.Parser.Parser.Start ( self)

Запускает процесс извлечения данных Firefox.

Метод выполняет полный цикл анализа данных Firefox в рамках одного кейса.

Последовательность работы:

  • проверяется соединение с результирующей БД;
  • извлекаются пути профилей Firefox;
  • для каждого профиля создаётся интерфейс чтения БД;
  • формируется объект Metadata;
  • запускаются стратегии извлечения данных:
    история, пароли, закладки, загрузки, расширения;
  • ожидается завершение всех задач;
  • результирующие базы данных сохраняются на диск.
Возвращает
Словарь вида: { <moduleName>: <имя результирующего файла БД> }
