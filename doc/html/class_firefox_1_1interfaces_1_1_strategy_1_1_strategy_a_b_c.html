Подробное описание

Абстрактный базовый класс стратегий обработки данных Firefox.

Класс StrategyABC определяет единый интерфейс для всех стратегий, предназначенных для извлечения и сохранения данных браузера Firefox.

Каждая конкретная стратегия обязана реализовать следующие методы:

  • read() — извлечение данных из источника;
  • write() — запись данных в результирующую базу данных;
  • execute() — полный цикл выполнения стратегии.

Базовый класс также предоставляет вспомогательные методы для инициализации SQLite базы данных, формирования служебных таблиц и получения временных меток.

Запускает полный цикл выполнения стратегии.

Как правило, метод:

  1. Создаёт необходимые таблицы;
  2. Инициирует чтение данных;
  3. Передаёт батчи на запись;
  4. Формирует служебные таблицы;
  5. Сохраняет результирующую базу данных на диск.
Аргументы
threadPoolПул потоков для асинхронного выполнения задач.
