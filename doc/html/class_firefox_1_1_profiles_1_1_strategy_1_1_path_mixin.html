Подробное описание

Миксин для получения пути к каталогу Firefox.

Предоставляет доступ к каталогу Firefox, расположенному в пользовательской директории APPDATA.

Используется стратегиями Firefox для формирования абсолютных путей к конфигурационным файлам.
