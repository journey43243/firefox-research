Подробное описание

Стратегия чтения и записи расширений Firefox.

Стратегия извлекает данные о расширениях из файла extensions.json, расположенного в каталоге профиля Firefox.

В обработку включаются только объекты с типом extension. Полученные данные преобразуются в структуру Extension и сохраняются в результирующую базу данных.

Выполняет полный цикл обработки расширений Firefox.

Аргументы
executorПул потоков для асинхронной записи данных.

Последовательно:

  1. Создаёт таблицу extensions.
  2. Считывает расширения из файла.
  3. Записывает данные с использованием пула потоков.
  4. Создаёт служебные таблицы и метаданные.
  5. Сохраняет результирующую базу данных.
