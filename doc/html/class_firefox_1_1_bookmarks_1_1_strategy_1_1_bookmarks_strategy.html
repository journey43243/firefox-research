Подробное описание

Стратегия чтения и записи закладок Firefox.

Класс считывает закладки из таблиц moz_bookmarks и moz_places и сохраняет их в результирующую базу данных.

Атрибуты: _logInterface Интерфейс логирования. _dbReadInterface Интерфейс чтения базы Firefox. _dbWriteInterface Интерфейс записи в выходную БД. _profile_id Идентификатор профиля Firefox.

Выполняет стратегию извлечения закладок Firefox.

Последовательно выполняет:

  1. Создание таблицы bookmarks.
  2. Чтение закладок из профиля Firefox.
  3. Асинхронную запись данных в выходную БД.
  4. Создание служебных таблиц и сохранение БД на диск.
Аргументы
executorПул потоков для асинхронной записи данных.

◆ read()

Firefox.Bookmarks.Strategy.BookmarksStrategy.read ( self)

Считывает закладки из базы Firefox.

Выполняет SQL-запрос к таблицам moz_bookmarks и moz_places. Обрабатываются только записи типа 1 (URL-закладки). Каждая запись преобразуется в объект Bookmark.

Возвращает
Список объектов Bookmark для текущего профиля.
