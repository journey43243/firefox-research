Подробное описание

Стратегия чтения и записи данных о загрузках Firefox.

Класс извлекает информацию о загрузках из таблиц moz_annos и moz_anno_attributes базы данных Firefox (places.sqlite).

Отбираются только записи, связанные с загрузками (атрибуты, содержащие подстроку downloads).

Данные читаются пакетами и записываются в результирующую базу данных с использованием многопоточной обработки.

Выполняет полный цикл обработки данных о загрузках Firefox.

Аргументы
executorПул потоков для асинхронной записи данных.

Последовательно:

  1. Создаёт таблицу downloads.
  2. Считывает данные о загрузках пакетами.
  3. Передаёт запись пакетов в пул потоков.
  4. Создаёт служебные таблицы и метаданные.
  5. Сохраняет результирующую базу данных на диск.

◆ read()

Generator[list[Download], None, None] Firefox.Downloads.Strategy.DownloadsStrategy.read ( self)

Считывает данные о загрузках из базы Firefox.

Выполняет SQL-запрос к таблице moz_annos с фильтрацией по атрибутам, связанным с загрузками (name LIKE 'downloads').

Данные возвращаются пакетами по 500 записей.
