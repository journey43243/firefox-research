Подробное описание

Стратегия чтения списка профилей Firefox.

Стратегия извлекает пути профилей Firefox из файла profiles.ini, расположенного в каталоге Firefox, и сохраняет их в результирующую базу данных.

Полученные пути используются другими стратегиями для обработки данных конкретных профилей браузера.

Запускает полный цикл извлечения профилей Firefox.

Последовательно:

  • считывает пути профилей из profiles.ini;
  • сохраняет их в результирующую базу данных;
